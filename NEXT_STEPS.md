## 🚀 立即执行的步骤

### 步骤 1: 停止所有服务

在三个终端中分别按 `Ctrl+C` 停止：
- 后端服务
- Agent 服务
- 前端服务

### 步骤 2: 清除前端缓存

**方法 A: 使用自动脚本（推荐）**
```bash
cd frontend
restart-dev.bat
```

**方法 B: 手动清除**
```bash
cd frontend
rmdir /s /q node_modules\.vite
npm run dev
```

### 步骤 3: 重启所有服务

**终端 1 - 后端**:
```bash
cd backend
npm run dev
```

**终端 2 - Agent**:
```bash
cd agent
python agent.py
```

**终端 3 - 前端**:
```bash
cd frontend
npm run dev
```

### 步骤 4: 清除浏览器缓存

1. 打开浏览器访问 http://localhost:5174
2. 按 `Ctrl + Shift + R` 进行硬刷新
3. 或者：
   - 打开开发者工具 (F12)
   - 右键点击刷新按钮
   - 选择 "清空缓存并硬性重新加载"

## ❌ 如果仍然有问题

### 问题 A: 控制台仍有 axios 错误

**解决方案**:
```bash
# 1. 完全删除 node_modules
cd frontend
rmdir /s /q node_modules
rmdir /s /q node_modules\.vite

# 2. 重新安装依赖
npm install

# 3. 启动服务
npm run dev
```

### 问题 B: 页面仍然空白

**检查步骤**:
1. 确认三个服务都在运行
2. 检查后端是否在 3000 端口
3. 检查前端是否在 5174 端口
4. 查看 `docs/前端空白页面问题排查.md` 获取详细排查步骤

## 💡 提示

### 开发时的最佳实践

1. **修改代码后**
   - Vite 会自动热重载
   - 如果没有生效，尝试硬刷新浏览器

2. **遇到奇怪问题时**
   - 首先清除 Vite 缓存
   - 然后硬刷新浏览器
   - 最后重启开发服务器

3. **保持服务运行**
   - 后端、Agent、前端三个服务都需要运行
   - 使用三个独立的终端窗口
   - 不要关闭终端
