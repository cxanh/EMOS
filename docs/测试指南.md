## 接口联调验证

以下示例均以当前默认端口 `50001` 为准。

### 1. 登录接口

```bash
curl -X POST http://localhost:50001/api/auth/login \
  -H "Content-Type: application/json" \
  -d '{
    "username": "admin",
    "password": "admin"
  }'
```

期望：

- 返回 `success: true`
- 返回 `data.token`

### 2. Agent 注册

注意：当前 Agent 注册接口受 `AGENT_API_TOKEN` 保护。

```bash
curl -X POST http://localhost:50001/api/agent/register \
  -H "Content-Type: application/json" \
  -H "X-Agent-Token: change_me_agent_token" \
  -d '{
    "node_id": "test-node-001",
    "hostname": "test-machine",
    "ip": "192.168.1.100",
    "display_name": "Test Node"
  }'
```

期望：

- 返回 `success: true`
- `data.registered` 为 `true`

### 3. Agent 上报指标

```bash
curl -X POST http://localhost:50001/api/agent/metrics \
  -H "Content-Type: application/json" \
  -H "X-Agent-Token: change_me_agent_token" \
  -d '{
    "node_id": "test-node-001",
    "metrics": {
      "cpu_usage": 45.2,
      "memory_usage": 50.0,
      "disk_usage": 68.1,
      "network_rx_bytes": 1024000,
      "network_tx_bytes": 512000
    },
    "timestamp": "2026-04-17T10:00:00.000Z"
  }'
```

期望：

- 返回 `success: true`

### 4. 获取最新监控数据

```bash
curl http://localhost:50001/api/metrics/latest/test-node-001
```

### 5. 获取历史数据

```bash
curl "http://localhost:50001/api/metrics/history?nodeId=test-node-001&startTime=2026-04-17T09:00:00Z&endTime=2026-04-17T10:00:00Z"
```

## WebSocket 验证

当前 WebSocket 路径为：

```text
ws://localhost:50001/ws/metrics
```

连接时需要 JWT Token，可通过 query string 传入：

```javascript
const ws = new WebSocket('ws://localhost:50001/ws/metrics?token=YOUR_JWT_TOKEN')

ws.onopen = () => {
  ws.send(JSON.stringify({
    type: 'subscribe',
    node_id: 'test-node-001'
  }))
}
```

检查点：

- 连接成功
- 可收到 `connected`
- 订阅后可收到 `subscribed`
- 有指标推送时可收到 `metrics`
