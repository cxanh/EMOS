## 📊 技术栈

### 后端
- **运行时**: Node.js 18+
- **框架**: Express 4.x
- **数据库**: Redis 7.x, InfluxDB 2.x
- **认证**: JWT (jsonwebtoken)
- **WebSocket**: ws
- **日志**: Winston
- **其他**: bcrypt, cors, dotenv

### 前端
- **框架**: Vue 3.x (Composition API)
- **语言**: TypeScript
- **路由**: Vue Router 4.x
- **状态管理**: Pinia 2.x
- **HTTP 客户端**: Axios
- **图表**: ECharts 5.x
- **构建工具**: Vite 5.x

### Agent
- **语言**: Python 3.8+
- **系统监控**: psutil
- **HTTP 客户端**: requests
- **配置**: PyYAML
