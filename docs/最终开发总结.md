## 🚀 快速启动

### 1. 启动后端
```bash
cd backend
npm install
npm run dev
```

### 2. 启动 Agent
```bash
cd agent
pip install -r requirements.txt
python agent.py
```

### 3. 启动前端
```bash
cd frontend
npm install
npm run dev
```

### 4. 访问系统
- 前端: http://localhost:5174
- 后端: http://localhost:3000
- 默认账号: admin / admin
