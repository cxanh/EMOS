## 📈 技术架构

### 技术栈

**前端**:
- Vue 3.5 (Composition API)
- TypeScript 5.x
- Vite 7.x
- Pinia (状态管理)
- Vue Router 4.x
- Axios (HTTP 客户端)
- ECharts 6.x (图表)

**后端**:
- Node.js 18+
- Express 5.x
- WebSocket (ws 8.x)
- JWT (认证)
- Winston (日志)

**数据库**:
- Redis 7.x (缓存)
- InfluxDB 2.x (时序数据)

**Agent**:
- Python 3.8+
- psutil (系统信息)
- requests (HTTP 客户端)
- pyyaml (配置管理)

**桌面**:
- Electron (跨平台)
