- **运行时**: Node.js 18+

- **数据库客户端**: 
  - redis (Node Redis)
  - @influxdata/influxdb-client

- **实时缓存**: Redis 7.x
- **时序数据库**: InfluxDB 2.x

- **框架**: Vue 3.5+
- **语言**: TypeScript 5.x
- **构建工具**: Vite 7.x
- **状态管理**: Pinia
- **路由**: Vue Router 4.x
- **HTTP 客户端**: Axios
- **图表库**: ECharts 6.x
- **UI 组件**: 自定义组件

### 桌面应用
- **框架**: Electron 最新稳定版

### 智能分析层
- **LLM 接口**: OpenAI API / 本地模型
- **库**: openai (Node.js SDK)

### 部署
- **容器化**: Docker, Docker Compose
- **反向代理**: Nginx
- **进程管理**: PM2

## 🚀 开发流程

### 1. 环境准备
```bash
# 安装 Node.js 18+
# 安装 Python 3.8+
# 安装 Redis
# 安装 InfluxDB
# 安装 Docker (可选)
```

### 2. 依赖安装
```bash
# 后端
cd backend
npm install

# 前端
cd frontend
npm install

# Agent
cd agent
pip install -r requirements.txt
```

### 3. 配置文件
- 复制 `.env.example` 为 `.env`
- 配置数据库连接信息
- 配置 LLM API 密钥

### 4. 启动服务
```bash
# 启动 Redis
redis-server

# 启动 InfluxDB
influxd

# 启动后端
cd backend
npm run dev

# 启动前端
cd frontend
npm run dev

# 启动 Agent
cd agent
python agent.py
```

### 5. 开发规范
- 使用 ESLint + Prettier 格式化代码
- 遵循 Git 提交规范 (Conventional Commits)
- 编写单元测试 (Jest / Vitest)
- 代码审查后合并
