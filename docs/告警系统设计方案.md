## 📋 功能需求

### 核心功能
1. **阈值告警** - CPU/内存/磁盘使用率超过阈值时触发
2. **通知方式** - 邮件、控制台日志、WebSocket 推送
3. **告警规则** - 可配置的告警规则
4. **告警历史** - 记录所有告警事件
5. **告警状态** - 待处理、已处理、已忽略

### 扩展功能（可选）
- 钉钉/企业微信通知
- 告警静默期
- 告警升级
- 告警分组

## 🗄️ 数据模型

### 1. 告警规则（Alert Rules）

存储在 Redis: `alert:rules`

```javascript
{
  id: "rule_001",
  name: "CPU 使用率过高",
  nodeId: "node001",  // "*" 表示所有节点
  metric: "cpu_usage",  // cpu_usage, memory_usage, disk_usage
  operator: "gt",  // gt(>), gte(>=), lt(<), lte(<=), eq(==)
  threshold: 80,  // 阈值
  duration: 60,  // 持续时间（秒），避免瞬时波动
  enabled: true,
  notifyChannels: ["email", "websocket"],  // 通知渠道
  createdAt: "2026-02-09T12:00:00Z",
  updatedAt: "2026-02-09T12:00:00Z"
}
```

### 2. 告警事件（Alert Events）

存储在 Redis: `alert:events:{eventId}`  
存储在 InfluxDB: measurement `alert_events`

```javascript
{
  id: "event_001",
  ruleId: "rule_001",
  ruleName: "CPU 使用率过高",
  nodeId: "node001",
  nodeName: "dev-machine",
  metric: "cpu_usage",
  currentValue: 85.5,
  threshold: 80,
  status: "active",  // active, resolved, ignored
  triggeredAt: "2026-02-09T12:00:00Z",
  resolvedAt: null,
  notified: true,
  message: "节点 dev-machine 的 CPU 使用率 (85.5%) 超过阈值 (80%)"
}
```

### 3. 告警历史（Alert History）

存储在 InfluxDB: measurement `alert_history`

```javascript
{
  time: "2026-02-09T12:00:00Z",
  eventId: "event_001",
  ruleId: "rule_001",
  nodeId: "node001",
  metric: "cpu_usage",
  value: 85.5,
  threshold: 80,
  status: "triggered"  // triggered, resolved, ignored
}
```

## 🔄 工作流程

### 1. 告警检测流程

```
1. Agent 上报数据 → 后端接收
2. 后端保存数据到 Redis/InfluxDB
3. 告警检测器检查数据
   ├─ 获取所有启用的告警规则
   ├─ 对比当前值与阈值
   ├─ 检查持续时间
   └─ 判断是否触发告警
4. 如果触发告警
   ├─ 创建告警事件
   ├─ 发送通知（邮件/WebSocket）
   └─ 记录到历史
5. 如果告警恢复
   ├─ 更新告警状态
   ├─ 发送恢复通知
   └─ 记录到历史
```

### 2. 告警通知流程

```
1. 告警触发
2. 获取通知渠道配置
3. 并行发送通知
   ├─ 邮件通知
   │   ├─ 构建邮件内容
   │   ├─ 发送邮件
   │   └─ 记录发送结果
   ├─ WebSocket 推送
   │   ├─ 构建推送消息
   │   ├─ 推送到前端
   │   └─ 前端显示通知
   └─ 控制台日志
       └─ 记录告警信息
4. 更新通知状态
```

## 💻 核心代码实现

### 1. 告警检测器（alertChecker.js）

```javascript
class AlertChecker {
  constructor(dataStore, alertService, notificationService) {
    this.dataStore = dataStore;
    this.alertService = alertService;
    this.notificationService = notificationService;
    this.checkInterval = 5000; // 5秒检查一次
    this.activeAlerts = new Map(); // 活动告警
  }

  start() {
    this.intervalId = setInterval(() => {
      this.checkAlerts();
    }, this.checkInterval);
  }

  async checkAlerts() {
    // 1. 获取所有启用的规则
    const rules = await this.alertService.getEnabledRules();
    
    // 2. 获取所有在线节点
    const nodes = await this.dataStore.getOnlineNodes();
    
    // 3. 检查每个规则
    for (const rule of rules) {
      const targetNodes = rule.nodeId === '*' 
        ? nodes 
        : nodes.filter(n => n.node_id === rule.nodeId);
      
      for (const node of targetNodes) {
        await this.checkRule(rule, node);
      }
    }
  }

  async checkRule(rule, node) {
    // 获取最新数据
    const metrics = await this.dataStore.getLatestMetrics(node.node_id);
    const currentValue = parseFloat(metrics[rule.metric]);
    
    // 判断是否触发
    const triggered = this.evaluateCondition(
      currentValue, 
      rule.operator, 
      rule.threshold
    );
    
    const alertKey = `${rule.id}_${node.node_id}`;
    
    if (triggered) {
      // 检查是否已经触发
      if (!this.activeAlerts.has(alertKey)) {
        // 新告警
        this.activeAlerts.set(alertKey, {
          triggeredAt: Date.now(),
          notified: false
        });
      }
      
      const alert = this.activeAlerts.get(alertKey);
      const duration = (Date.now() - alert.triggeredAt) / 1000;
      
      // 检查持续时间
      if (duration >= rule.duration && !alert.notified) {
        // 触发告警
        await this.triggerAlert(rule, node, currentValue);
        alert.notified = true;
      }
    } else {
      // 告警恢复
      if (this.activeAlerts.has(alertKey)) {
        await this.resolveAlert(rule, node, currentValue);
        this.activeAlerts.delete(alertKey);
      }
    }
  }

  evaluateCondition(value, operator, threshold) {
    switch (operator) {
      case 'gt': return value > threshold;
      case 'gte': return value >= threshold;
      case 'lt': return value < threshold;
      case 'lte': return value <= threshold;
      case 'eq': return value === threshold;
      default: return false;
    }
  }

  async triggerAlert(rule, node, currentValue) {
    // 创建告警事件
    const event = await this.alertService.createEvent({
      ruleId: rule.id,
      ruleName: rule.name,
      nodeId: node.node_id,
      nodeName: node.hostname,
      metric: rule.metric,
      currentValue,
      threshold: rule.threshold,
      status: 'active'
    });
    
    // 发送通知
    await this.notificationService.send(event, rule.notifyChannels);
  }

  async resolveAlert(rule, node, currentValue) {
    // 更新告警状态
    await this.alertService.resolveEvent(rule.id, node.node_id);
    
    // 发送恢复通知
    await this.notificationService.sendRecovery({
      ruleName: rule.name,
      nodeId: node.node_id,
      nodeName: node.hostname,
      metric: rule.metric,
      currentValue
    });
  }
}
```

### 2. 通知服务（notificationService.js）

```javascript
class NotificationService {
  constructor(emailSender, websocketServer) {
    this.emailSender = emailSender;
    this.wss = websocketServer;
  }

  async send(event, channels) {
    const promises = [];
    
    if (channels.includes('email')) {
      promises.push(this.sendEmail(event));
    }
    
    if (channels.includes('websocket')) {
      promises.push(this.sendWebSocket(event));
    }
    
    await Promise.allSettled(promises);
  }

  async sendEmail(event) {
    const subject = `[告警] ${event.ruleName}`;
    const html = `
      <h2>系统告警通知</h2>
      <p><strong>告警规则:</strong> ${event.ruleName}</p>
      <p><strong>节点:</strong> ${event.nodeName} (${event.nodeId})</p>
      <p><strong>指标:</strong> ${event.metric}</p>
      <p><strong>当前值:</strong> ${event.currentValue}%</p>
      <p><strong>阈值:</strong> ${event.threshold}%</p>
      <p><strong>触发时间:</strong> ${event.triggeredAt}</p>
      <p><strong>消息:</strong> ${event.message}</p>
    `;
    
    await this.emailSender.send({
      to: process.env.ALERT_EMAIL,
      subject,
      html
    });
  }

  async sendWebSocket(event) {
    this.wss.broadcast({
      type: 'alert',
      data: event
    });
  }
}
```
