# 根目录历史材料

本目录存放从项目根目录迁出的历史性文档和材料。

迁移目的：

- 清理项目根目录
- 保留历史记录
- 避免阶段性报告与当前有效文档混在一起

这些文件通常包括：

- 功能完成总结
- 修复记录
- 快速指南
- 阶段路线图
- 测试快照
- 课题材料

它们仅用于追溯，不代表当前项目基线。
