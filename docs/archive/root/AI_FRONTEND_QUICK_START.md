## 🐛 常见问题

### 问题1: AI分析失败

**错误信息**: "AI service not enabled" 或 "Ollama API error"

**解决方案**:
```bash
# 1. 检查Ollama是否运行
ollama list

# 2. 检查后端配置
cat backend/.env | grep LLM

# 3. 重启Ollama
ollama serve

# 4. 测试AI服务
cd backend
node test-ai-standalone.js
```

### 问题2: 页面空白或报错

**解决方案**:
```bash
# 1. 检查浏览器控制台错误
# 按F12打开开发者工具

# 2. 检查前端编译错误
cd frontend
npm run dev

# 3. 清除缓存重新编译
rm -rf node_modules/.vite
npm run dev
```

### 问题3: 加载时间过长

**原因**: Ollama模型响应较慢

**解决方案**:
- 使用更小的模型（如 llama2:7b）
- 增加超时时间
- 检查系统资源（CPU/内存）

### 问题4: 分析结果不准确

**原因**: 模型理解能力有限

**解决方案**:
- 使用更强大的模型（如 llama2:13b）
- 优化提示词
- 提供更多上下文数据
