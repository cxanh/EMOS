## 📊 技术实现

### 后端技术栈
- **语言**: JavaScript (Node.js)
- **框架**: Express
- **数据库**: Redis + InfluxDB
- **数据聚合**: 1小时粒度
- **存储策略**: 30天TTL

### 前端技术栈
- **框架**: Vue 3
- **语言**: TypeScript
- **HTTP客户端**: Axios
- **UI设计**: 响应式卡片布局

### 数据流程
```
用户请求 → 后端API → 查询InfluxDB历史数据 
→ 数据聚合和统计 → 保存到Redis → 返回报表
```

### 数据存储
```
Redis数据结构:
├── report:{reportId} (Hash) - 报表元数据
└── reports (Set) - 报表ID列表

InfluxDB:
└── 历史监控数据（查询源）
```

### 3. 生成第一份报表

1. 点击"生成报表"按钮
2. 选择报表类型（建议先选"日报"）

5. 等待生成完成，自动打开详情

