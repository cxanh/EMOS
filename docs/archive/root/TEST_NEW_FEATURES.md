### 1. 多用户管理系统测试

#### 后端测试
```bash
cd backend
node test-user-management.js
```

**预期结果**:
- ✅ 12个测试用例全部通过
- ✅ 成功率100%

#### 前端测试

1. **访问用户管理页面**
   - URL: http://localhost:5174/users
   - 要求：管理员账户登录

2. **创建用户**
   - 点击"创建用户"按钮
   - 填写用户信息
   - 选择角色
   - 点击"确定"
   - 验证：用户列表中出现新用户

3. **编辑用户**
   - 点击用户行的"编辑"按钮
   - 修改用户信息
   - 点击"确定"
   - 验证：用户信息已更新

4. **禁用/启用用户**
   - 点击用户行的"禁用"按钮
   - 验证：用户状态变为"禁用"
   - 点击"启用"按钮
   - 验证：用户状态变为"正常"

5. **删除用户**
   - 点击用户行的"删除"按钮
   - 确认删除
   - 验证：用户从列表中消失
