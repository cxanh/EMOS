## 🎯 使用建议

### 邮件通知

1. **推荐配置**:
   ```env
   SMTP_HOST=smtp.gmail.com
   SMTP_PORT=587
   SMTP_SECURE=false
   ```

2. **注意事项**:
   - 必须使用应用专用密码，不是邮箱登录密码
   - Gmail需要先启用两步验证
   - 检查垃圾邮件文件夹

3. **替代方案**:
   - 如果Gmail连接不稳定，可以使用QQ或163邮箱
   - 国内邮箱通常更稳定

### 钉钉通知

1. **推荐配置**:
   - 安全设置: 加签
   - 配置DINGTALK_SECRET

2. **注意事项**:
   - 确保复制完整的密钥（包括SEC前缀）
   - 机器人不要被禁用
   - 检查群聊是否正常

## 🐛 问题排查

### 邮件通知问题

#### 问题1: 连接超时
- **原因**: 网络问题或防火墙阻止
- **解决**: 
  - 检查网络连接
  - 尝试使用VPN
  - 改用国内邮箱

#### 问题2: 认证失败
- **原因**: 密码错误
- **解决**:
  - 确认使用应用专用密码
  - 重新生成密码
  - 检查两步验证

### 钉钉通知问题

#### 问题1: 签名错误
- **原因**: 密钥配置错误
- **解决**:
  - 检查DINGTALK_SECRET
  - 确认包含SEC前缀
  - 确认安全设置是"加签"

#### 问题2: 关键词不匹配
- **原因**: 安全设置是"自定义关键词"
- **解决**:
  - 改为"加签"安全设置
  - 或添加关键词"告警"
