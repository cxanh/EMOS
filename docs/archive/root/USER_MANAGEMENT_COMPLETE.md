## 🎯 功能特性

### 1. 三级角色权限体系

| 角色 | 权限 |
|------|------|
| **管理员 (Admin)** | 所有功能 + 用户管理 |
| **操作员 (Operator)** | 监控 + 告警配置 |
| **查看者 (Viewer)** | 仅查看监控数据 |

### 2. 完整的用户生命周期管理

```
创建 → 激活 → 使用 → 禁用 → 启用 → 删除
```

### 3. 安全特性

- ✅ bcrypt密码加密
- ✅ JWT Token认证
- ✅ 密码强度验证
- ✅ 用户状态控制
- ✅ 权限级别验证
- ✅ 操作审计日志

### 4. 数据验证

**用户名**:
- 长度: 3-20字符
- 格式: 字母、数字、下划线
- 唯一性检查

**密码**:
- 最小长度: 6字符
- bcrypt加密存储

**邮箱**:
- 格式验证
- 唯一性检查（可选）

### 5. 操作保护

- ❌ 无法删除自己的账户
- ❌ 无法删除最后一个管理员
- ❌ 禁用用户无法登录
- ✅ 管理员可重置任何用户密码

## 📊 技术实现

### 后端技术栈
- **语言**: JavaScript (Node.js)
- **框架**: Express
- **数据库**: Redis
- **认证**: JWT + bcrypt
- **日志**: Winston

### 前端技术栈
- **框架**: Vue 3
- **语言**: TypeScript
- **状态管理**: Pinia
- **路由**: Vue Router
- **HTTP客户端**: Axios

### 数据存储
```
Redis数据结构:
├── user:{userId} (Hash) - 用户信息
├── users (Set) - 用户ID列表
├── username:{username} (String) - 用户名映射
└── email:{email} (String) - 邮箱映射
```
