# 分布式感知运维系统 - 项目总览

## 📌 项目信息

**项目名称**: EOMS (Enterprise Operations Monitoring System) - 分布式感知运维系统  
**项目类型**: 本科毕业设计  
**技术栈**: Vue 3 + Node.js + Python + Redis + InfluxDB  
**开发模式**: 前后端分离 + 多端支持  
**文档创建日期**: 2026-02-06

---

## 🎯 项目目标

构建一个支持多节点监控、实时数据展示、历史数据分析和智能运维建议的分布式监控系统。

### 核心功能

1. **多节点系统资源监控** - 实时采集 CPU、内存、磁盘、网络数据
2. **多端数据互通** - Web、桌面端、移动端统一访问
3. **实时展示 + 历史分析** - 实时监控 + 时序数据查询
4. **大模型辅助运维** - LLM 智能分析和建议

## 🛠️ 技术栈

### 采集层
- Python 3.8+
- psutil (系统信息采集)
- requests (HTTP 请求)
- pyyaml (配置管理)

- Node.js 18+
- Express 5.x (Web 框架)

### 数据支撑层
- Redis 7.x (实时数据缓存)
- InfluxDB 2.x (时序数据存储)

### 前端层
- Vue 3.5+ (框架)
- TypeScript 5.x (类型系统)
- Vite 7.x (构建工具)
- Pinia (状态管理)
- Vue Router 4.x (路由)
- Axios (HTTP 客户端)
- ECharts 6.x (图表)

### 桌面应用
- Electron (跨平台桌面应用)

### 智能分析层
- OpenAI API / 本地 LLM

### 部署
- Docker + Docker Compose
- Nginx (反向代理)
- PM2 (进程管理)

## 🚀 快速开始

### 方式一: 本地开发

```bash
# 1. 安装依赖
cd backend && npm install
cd frontend && npm install
cd agent && pip install -r requirements.txt

# 2. 启动 Redis 和 InfluxDB
redis-server
influxd

# 3. 启动服务
cd backend && npm run dev    # 终端 1
cd frontend && npm run dev   # 终端 2
cd agent && python agent.py  # 终端 3

# 4. 访问应用
# http://localhost:5174
```

### 方式二: Docker 部署

```bash
# 一键启动所有服务
docker-compose up -d

# 查看服务状态
docker-compose ps

# 访问应用
# http://localhost:80
```
